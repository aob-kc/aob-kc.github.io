Order

(一)

del : 讓文字增加刪除線效果

sup : 字體較小、位置偏上

sub : 字體較小、位置偏下

ul : 前綴為圖示的列表清單

ol : 前綴為數字的列表清單


pre : 讓文字隨著文章斷行而斷行，不需下br進行斷行

ins : 文字加上底線

bdo : 添加屬性dir="rtl or ltr"

此為bdo示範效果

blockquote : 塊引用。自動生成padding效果


(二)

nav : navbar區塊

header : 章節頭部

footer : 章節尾部

main : 章節當中的主要內容

title : 該屬性添加後，當滑鼠移至該物件，會出現輸入於title內的文字


dl/dt/dd : 建立清單/項目/該項目的描述

figure : 代表與文檔有關的圖例

figcaption : 針對< figure>做文字說明

video : 放置影片。
需要添加靜音屬性才能自動撥放

audio : 放置純音檔


(三)

table/tr/th/td : 建立表格/列/首列標題/內容

caption : 表格標題，放置於tr、th、th之上，table之下

colgroup : 內部放col的tag，可對此添加span="影響欄數"style="作用效果"

textarea : 功用同input，只是此可設定寬高、欄列限制

maxlength可限制textarea當中的最大文字輸入限度

meter : 設定min、max訂定範圍值，再給透過value給予其比例

另可設定low、high屬性，讓value低於或超過時改變顏色顯示警告

select : 下拉選單

optgroup/option : 兩者皆出現於下拉選單內，前者作為標題區分後者，後者即為區分後的子項目

optgroup內添加label屬性，即可顯示標題名稱
option內添加value給予值

特殊字符

& nbsp; : 空格效果(需把&與n中間空格拔掉


新視窗開啟錨點連結

< a href="" target="_blank" >


連結點擊下載

於< a>內添加download屬性，且href放置欲下載的檔案，download填的是下載時的檔案名稱


(一)

outline : 物件外框邊線，和border不同，是向外生長

text-decoration : 讓"線"出現在文字的上、中、下

上:overline/中:line-through/下:ubderline

text-transform : 控制英文大小寫設定

全大寫:uppercase/全小寫:lowercase/僅單字字首大寫:capitalize

text-indent : 使段落開頭內縮指定數值

word-spacing : 單字與單字間的間距調整

font-style : italic ←使字變成斜體


組合器

後代選擇器(space[空格鍵]) :
例 : #a .b

子選擇器(>) :
例 : div > p

div子層內的所有p進行選取，針對子層進行作用

相鄰選擇器(+) :
例 : div + p

選取div後的p。不須相鄰，不可夾著其他tag

兄弟選擇器(~) :
例 : div ~ p

選取div後的所有p。通父層才可進行作用

writing-mode: vertical-lr;
讓文字呈現垂直排版


bg-color漸層
